Fill To - 개인정보 처리방침

1. 개인정보 수집 및 이용 목적

Fill To 앱은 사용자의 개인정보를 수집하지 않습니다. 모든 기분 기록 데이터는 사용자의 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 수집하는 개인정보 항목

수집하는 개인정보: 없음

  • Fill To는 어떠한 개인정보도 수집하지 않습니다
  • 기분 기록, 메모 등 모든 데이터는 기기 내부에만 저장됩니다
  • 네트워크 연결이나 인터넷 접근이 필요하지 않습니다

3. 개인정보 보관 및 삭제

모든 데이터는 사용자의 기기에만 저장되므로:

  • 사용자가 앱을 삭제하면 모든 데이터가 함께 삭제됩니다
  • 데이터 백업 및 복원 기능을 통해 사용자가 직접 데이터를 관리할 수 있습니다
  • 개발자는 사용자의 데이터에 접근할 수 없습니다

4. 개인정보 보호 조치

  • 모든 데이터는 기기 내부 SQLite 데이터베이스에 저장됩니다
  • 외부 서버나 클라우드 서비스를 사용하지 않습니다
  • 네트워크 권한을 요구하지 않습니다
  • 광고나 분석 도구를 사용하지 않습니다

5. 권한 사용

Fill To는 다음 권한만을 사용합니다:

  • 알림 권한: 기분 기록 리마인더 알림을 위해 사용됩니다 (선택사항)
  • 저장소 접근: 데이터 백업/복원 파일 저장을 위해 사용됩니다 (선택사항)

Fill To - Privacy Policy

1. Purpose of Personal Information Collection and Use

The Fill To app does not collect any personal information. All mood tracking data is stored only on the user's device and is never transmitted to external servers.

2. Personal Information Items Collected

Personal Information Collected: None

  • Fill To does not collect any personal information
  • All data including mood records and memos are stored only on the device
  • No network connection or internet access is required

3. Personal Information Storage and Deletion

Since all data is stored only on the user's device:

  • All data is deleted when the user uninstalls the app
  • Users can manage their data directly through backup and restore features
  • Developers cannot access user data

4. Personal Information Protection Measures

  • All data is stored in a local SQLite database on the device
  • No external servers or cloud services are used
  • No network permissions are required
  • No advertising or analytics tools are used

5. Permissions Used

Fill To uses only the following permissions:

  • Notification Permission: Used for mood tracking reminder notifications (optional)
  • Storage Access: Used for saving data backup/restore files (optional)

Fill To - プライバシーポリシー

1. 個人情報の収集・利用目的

Fill Toアプリは個人情報を収集しません。すべての気分記録データはユーザーのデバイス内部にのみ保存され、外部サーバーに送信されることはありません。

2. 収集する個人情報項目

収集する個人情報：なし

  • Fill Toはいかなる個人情報も収集しません
  • 気分記録、メモなどすべてのデータはデバイス内部にのみ保存されます
  • ネットワーク接続やインターネットアクセスは必要ありません

3. 個人情報の保管・削除

すべてのデータはユーザーのデバイスにのみ保存されるため：

  • ユーザーがアプリを削除すると、すべてのデータも一緒に削除されます
  • データバックアップ・復元機能を通じてユーザーが直接データを管理できます
  • 開発者はユーザーのデータにアクセスできません

4. 個人情報保護措置

  • すべてのデータはデバイス内部のSQLiteデータベースに保存されます
  • 外部サーバーやクラウドサービスは使用しません
  • ネットワーク権限を要求しません
  • 広告や分析ツールは使用しません

5. 権限の使用

Fill Toは以下の権限のみを使用します：

  • 通知権限：気分記録リマインダー通知のために使用されます（任意）
  • ストレージアクセス：データバックアップ・復元ファイルの保存のために使用されます（任意）
